XSS 登录练习挑战

第一关：通过XSS注入实现登录

挑战说明

1

理解挑战目标

本关卡中，您需要通过XSS注入实现登录。系统不验证密码，但要求用户名中包含特定的JavaScript代码。

2

成功条件

在用户名输入框中注入包含以下代码的字符串：

alert(1)

成功注入后点击登录按钮，您将进入第二关

3

尝试注入

在用户名输入框中尝试输入包含JavaScript代码的字符串：

<script>alert(1)</script>

或者尝试其他XSS注入方式

4

进入下一关

成功登录后，您将被重定向到论坛页面，在那里可以进行第二关的挑战
